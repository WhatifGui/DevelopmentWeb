Paragráfos e quebras de linhas

Indenpendentemente da forma que for escrita dentro da tag se é quebrando linha ou não, no site ele ficará sequencialmente, mostrando diferente que é no código.

Um segundo paragráfo, para exemplificar a quebra de linha dentro do texto.
Continuando o texto embora sequencialmente no código no site há a quebra de linha, devido a tag br

A tag br, significa Break row = Quebra de linha

Para em caso de querer escrever o sinal de maior e menor para mostrar dentro do site as tags terá que escrever <p> e </p>
